# DAPP NODE API

This API is created to register all **NFTs** build on our platform. 

To run it localy.

`git clone https://github.com/Filip3Dev/dapp-api.git`

then

`npm run install`

then you should create a `.env` file to add a `MONGODB_URI`.

finally

`netlify dev`
